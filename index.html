C. What? Why?

C is a general-purpose computer programming language. C is a lower-level language which means that it allows you to mess with the hardware , but needs a compiler and is harder to shoot you foot off. Learning C is important because most languages are based on it and C is very powerful.

C isn't that beginner friendly and has no garbage collector, so you must mange your own memory. If that sounded weird, don't worry about it. I will explain in more in depth latter

How will you teach me?

This is split into 4 sections Liftoff, Repair, New Tools, and Final project. To follow along open up an IDE (And install the C compiler) or go to an online compiler like this one. All lessons are written by me and I try to explain everything as good as I can.

Unpacking

Where to begin? The first line, #include <stdio.h>, includes/imports a library which is a collection of pre-defined functions. This specific libraries, stdio, stands for Standard Input Output and allows you to use output and input vaules using the console

Chars

On the second line we use the keyword Char. What does this mean? Well you probably guessed it. It stands for character. But what we do after the keywords matters alot. Because C has no string type, we must create an array of Chars and assign a Char to each index. We could do this with pointers, but we will get to that latter.

Int

On the third line we declare a Int data-type. This allows us to use an integers as a variable. Why can't we just put var in front of everything? C is strongly typed which means we must annotate types.

Float vs Double

I know your probably looking at this and thinking, what the big deal between float and double? A floating point and a double are decimal point numbers. But, Floats only have 7 digits of precision while doubles have 15.

Everything Else

After all the data types, we see a function. This function is different from normal functions. It is the main function. Your code starts executing in the main function instead of the global spoce. The void means it shouldn't take any input. The return 0; means everything worked. If it is 1 then it is bad.

Functions

A function is defined with the return type, then there is the name, and then the parameters and the block of code. The syntax is parameters between braces and the block of code between curly braces. type name(parameter){ block of code }

If/Else

Inside the max function, we use a if/elsecondition. The if and else syntax in C is the same in Javascript. There is also else if condition which is also the same as Javascript.

For loops

A For loop is slightly different in C then in JS. To start a For loop we first must initialize are indexs. Unlike in JS, we do not initialize a index on the same line. The only other thing that should jump out to you is the %d and /n in the print statement. The %d tells the computer to throw in the i where the %d is. The /n means new line.

Pointers

A pointer is a variable that stores the memory address of another variable as its value. A pointer variable points to a data type (like int ) of the same type, and is created with the * operator.

Unpacking the rest

Now that we know about pointers we can begin to explain what's happening here. The pointy variable stores an integer . But, we want to know where it is stored on the hardware. So we know we want to use pointers in some way, but how? If we print &pointy it will use the pointer and print the adress. Now we are ready for the rest.
